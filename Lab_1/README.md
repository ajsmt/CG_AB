# Описание
Реализация 3 цветовых моделей (RGB, CMYK и HSV)
## Возможности
- Управление:
    - Слайдеры 
    - Поля для ввода
    - Палитра
- Автоматический пересчет цветов
- Предпросмотр текущего цвета

---
## Используемые техноголии
- С++ (17 стандарт)
- Графический интерфейс: Dear ImGui
    - GLFW
    - OpenGL 
- Cmake
