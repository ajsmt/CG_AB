# Лабораторные задания по курсу "Компьютерная графика"
---
Все лаборатоные работы находятся в директориях - Lab_{номер работы}

Описания и используемые техноголии для каждой лабраторной работы находятся в README.md каждой лабы
---
## Сборка
Для сборки проекта можно использовать скрипт build.cmd, указывая какую лаюораторную работу собирать
Пример
```
./build.cmd Lab_1
```
После чего в папке build будет находится собранная работа
